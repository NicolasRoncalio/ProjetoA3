# ProjetoA3
Trabalho A3 da faculdade UNISOCIESC

Este projeto é uma inplementação de uma tela de cadastro para uma academia onde o usuario podera inserir os dados do aluno(nome, cpf, email, data de nascimento e telefone) e podera adicionar treinos para o aluno com a ajuda de uma tabela para visualização das informações do aluno e do treino.

1° passo: Digite as informações do aluno nos campos requiridos e clique em "CADASTRAR", se alguma irformação foi digitada errada alterea no campo designado e clique em "ATUALIZAR", se as imformaç~eos do aluno não são mais desejadas clique em "DELETAR".

2° passo: Após a adição do aluno para adicionar um treino a este alano coloque o cpf do aluno no campo designado e clique em "TREINOS", você sera redirecionado a uma nova tela para cadastrar treinos a este aluno.

3° passo: digite o cpf do aluno desejado e digite os campos abaixo(tipo de treino, duração em min, data de inicio e a descrição do treino) e clique em "INSERIR" para que o treino seja cadastrado, se alguma informação sobre o treino estiver errada corrija el clique em "ATUALIZAR".

NOTA: ao clicar nas informações do aluno os campos serão preenchidos automaticamente mas atenção a maneira que data esta inserida ja que ela deve ser escrita da seguinte forma (dd/mm/yyyy) mas quando é trazida da tabela ela esta no formato (yyyy-mm-dd).
